# 问题

最近在接MSDK，调用微信/手Q分享接口

    WGSendToQQ

及

    WGSendToWeixin

接口时均报错

    WGSendToQQ function not permission

    WGSendToWeixin function not permission

# 解决

询问MSDK官方后得到的答复是

> 联运的游戏是不具有这个功能的权限的，也是无法申请的
